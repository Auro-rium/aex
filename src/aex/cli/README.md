# CLI - v2.0

Typer-based operator interface.

Main command groups:
- `aex daemon` - start/stop/status
- `aex agent` - create/inspect/delete/rotate/state
- `aex models` - config reload
- `aex plugin` - plugin management
- `aex replay` - hash-chain + balance replay checks

CLI controls the same governance primitives used by HTTP paths.
